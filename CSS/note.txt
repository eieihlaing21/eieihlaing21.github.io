-Box Model
  -Border : width style color; 
-Margin
  -margin(one value) 
   -margin : all(top/right/bottom/left);
  -margin (two values)
   -margin :(top&bottom)/(left&right); 
  -margin (three values)
   -margin : top (right & left) bottom;
  -margin (four values)
   -margin : top right bottom left;
-Padding
  -padding(one value) 
   -padding : all(top/right/bottom/left);
  -padding (two values)
   -padding :(top&bottom)/(left&right); 
  -padding (three values)
   -padding : top (right & left) bottom;
  -padding (four values)
   -padding : top right bottom left;

-Display
 -Inline
   -doesn't work css properties in inline elements
   -display as a line
   -works margin and padding in inline elements
 -Block
   -works css properties in block elements
   -doesn't display as a line
 -Inline-Block
   -works css properties 
   -display as a line
